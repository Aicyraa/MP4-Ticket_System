# MP4-Ticket_System
Our group entry for DASTRUC 

### Project Setup
```bash

   # Step 1 (Fork the project)
      Fork button is located above "About" text

   # Step 2 (Clone the project)
      git clone git@github.com:Aicyraa/MP4-Ticket_System.git
      cd ./MP4-Ticket_System

   # Step 3 (Creating .venv)
      
      # Create a .venv file
      python3 -m venv .venv
      
      # Activate .venv first before you do anything!!

      # Linux
      source .venv/bin/activate

      # Windows (open CMD then go to the project's directory)
      .venv\Scripts\Activate

   # Step 4 (install packages)

      pip3 or pip install -r requirements.txt
```
